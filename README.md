# Multi-Agent AI System for Hotel Reservations

A sophisticated multi-agent AI system for managing hotel reservations with intelligent booking, customer service, and inventory management.

## Features
- Intelligent booking agent
- Customer service chatbot
- Inventory management
- Dynamic pricing optimization

## Tech Stack
- Multi-Agent AI Framework
- Python
- Natural Language Processing

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
